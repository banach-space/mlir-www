18 // Insert resharding spmdization of the value `sourceShardValue`
19 // from sharding `source` to sharding `target`.
20 // `sourceShardValue` is the already sharded value according to `source`.
21 //
22 // Example
23 //
24 // ```mlir
25 // mesh.mesh @mesh_1d(shape = 2)
26 // ...
27 // %1 = mesh.shard %0 to <@mesh_1d, [[0]]> : tensor<2xi8>
28 // %2 = mesh.shard %1 to <@mesh_1d, [[]]> annotate_for_users: tensor<2xi8>
29 // ```
30 //
31 // Will result in

33 // ```mlir

36 // ```
